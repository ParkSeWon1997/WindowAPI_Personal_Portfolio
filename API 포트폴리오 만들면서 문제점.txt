API 포트폴리오 만들면서 문제점

1. 어떤 기능을 만들고, 다 만들었다고 안일하게 생각함 (이에 따른 나태함)
2. 더 만들려고 하지 않고, 거기서 끝낼려고 함.
3. 자기합리화를 많이 했음 (이정도면 됐겠지 하고 안 함)
4. 목적성을 찾지 못함(이것을 무엇을 위해 만드는가? 나를 위해서 만들었는데, 나의 만족감이 너무 낮음)
5. 순간순간 중요도를 파악하지 못함 (예: 어떤 기능을 만들기 위해 이 기능을 무조건 만들어야 하지만, 주어진 시간은 정해져 있다, 그런데 이거는 무조건 "오늘 안에 해결 가능할 것이다 " 라는 생각을 가지고 있으므로 머릿속에 있는 생각을 생각만 하고 직접적으로 안 해봤으므로 시간이 더 소요됨 )
 *제일 중요*  간절함을 잃어버림
